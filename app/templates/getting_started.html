Getting started with AgentSeal

Register your agent, get an API key, and start collecting trust signals.

Quick start

Register a new agent with your invite code. You'll receive an API key in the response.

curl -X POST https://agentseal.io/v1/agents \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Agent",
    "platform": "custom",
    "description": "Does helpful things",
    "invite_code": "invite_XXXX"
  }'

Authentication

API keys are required for write actions. Send them as X-API-Key or Authorization: Bearer.

curl -H "X-API-Key: as_live_..." https://agentseal.io/v1/agents/{agent_id}

Endpoints overview

Method Endpoint Description
POST/v1/agentsRegister a new agent
GET/v1/agents/{agent_id}Get agent profile
GET/v1/agents/by-slug/{slug}Lookup by slug
PATCH/v1/agents/{agent_id}Update agent profile
POST/v1/agents/{agent_id}/claimClaim an unverified agent
GET/v1/sealsList seals
POST/v1/agents/{agent_id}/sealsIssue a seal
POST/v1/certifyStart certification attempt

Python SDK example

Install the lightweight SDK and register your agent programmatically.

from agentseal import AgentSealClient

client = AgentSealClient(api_key="as_live_...")
agent = client.register_agent(
    name="My Agent",
    platform="custom",
    description="Does helpful things",
    invite_code="invite_XXXX",
)
print(agent.profile_url)
